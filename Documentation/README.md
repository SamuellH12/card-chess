
# Card Chess // Chess dinâmico

Um jogo baseado no xadrez, mas com um dinamismo de deck de cartas, expandindo o jogo clássico do xadrez com novas peças e uma gameplay mais dinâmica.

- Cada peça, clássica ou nova, vai ser representada por uma carta. 

- No início do jogo o jogador tem acesso a um deck básico que pode posicionar no tabuleiro antes do jogo começar.

- Durante o jogo haverá um deck comum de cartas, do qual os jogadores podem comprar cartas durante o jogo para conseguir novas peças. A cada X lances, um jogador escolhe entre fazer uma jogada ou receber uma carta. 

- Cada jogador pode realizar 1 (ou 2?) movimentos por turno. Podendo ser, mover peças, posicionar novas cartas, etc.

- Além de cartas de peças, haverão cartas de ação com dinâmicas para adicionarmos no jogo. Alguns possíveis exemplos são:
    - Congele a rainha inimiga por 3 turnos
    - Seu rei fica protegido de cheques por 4 turnos
    - Todos seus peões podem andar 3 casas na primeira vez
    - Alterar características do grid (como formato)
    - Faça seu adversário perder todas as cartas
    - Promova um peão independente de onde esteja


- Jogabilidade multiplayer local (sem opção online).

Trabalhar principalmente nas variações de ações e peças para permitir diferentes estratégias de jogo.
